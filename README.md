# Jiffy backup to Toggl import converter

This package converts time-tracking data from Jiffy (Android backup format) into Toggl-compatible CSV files, allowing for easy import into Toggl.

> [!IMPORTANT]
> This project lacks tests; do not rely on it for critical data.

## Status

The library has functionality for:

- Parsing Jiffy data.
- Formatting Toggle data.

## Contributing / Contact

If you also want to export from Jiffy to other formats or import from other formats into Toggl, feel free to reach out or contribute!
